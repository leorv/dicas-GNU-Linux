# Princípios Unix

Vamos falar de um conjunto de sistemas, chamado *Unix Like*.

Ken Thompson e Dennis Ritchie (1965 -> Bell labs, MIT, AT&T).

Sistema chamado Multics, mas a junção das empresas não deu certo. Dissolução.

Em 1969 eles começaram a desenvolver um sistema chamado Unics com "c", porém, depois foi Unix. Escreveram em *Assembly*.

1973 -> Reescreveram usando a linguagem C.

Pra ter ideia de quem é Ken Thompson, ele foi o criador da linguagem B. Hoje, um dos criadores da linguagem Go.

## Ideias

Os programas devem fazer uma única coisa, mas fazer bem feito.

Vários programas trabalham juntos, conversam entre si.
